Trauma-Informed UI Atoms

Basic building blocks of our trauma-informed design system. Each component prioritizes user safety, predictability, and empowerment.

Emergency Exit Button

Emergency Exit Button

Purpose & Considerations

  • • Provides immediate escape from potentially triggering content without confirmation dialogs
  • • Always visible and accessible in the bottom-right corner of the viewport
  • • High contrast rhodochrosite-rose color ensures visibility while signaling urgency
  • • Single-click activation respects user's need for immediate exit

Size Variations

  • • Small (sm): Compact 40x40px touchpoint, ideal for tight spaces while maintaining accessibility
  • • Medium (md): Default 44x44px touchpoint, optimal for general use and accessibility standards
  • • Large (lg): Enhanced 48x48px touchpoint, improved visibility for important actions
  • • Extra Large (xl): Maximum 52x52px touchpoint, highest visibility for critical interactions

Responsive Behavior

  • • Mobile (< 768px): Small size (sm) for space efficiency while maintaining accessibility
  • • Tablet (768px - 1023px): Medium size (md) for optimal touch target
  • • Desktop (≥ 1024px): Large size (lg) for enhanced visibility
  • • Text label hides on smallest screens but remains accessible via aria-label

Primary Button

Primary Button

Purpose & Considerations

  • • Used for main actions and positive progression through content
  • • Calming chrysocolla-green color promotes safety and encouragement
  • • Gentle transitions and hover effects prevent startling interactions
  • • Clear visual feedback maintains predictability and user confidence
  • • Responsive text and icon sizing, accessible min-h/width (44px+), and dynamic label

Size Variations

  • • Small (sm): min-h/min-w 44px, text-small, icon 16px, label: "Continue"
  • • Medium (md): min-h 44px, text-base, icon 20px, label: "Continue Learning"
  • • Large (lg): min-h 52px, responsive text-h4, icon 24px, label: "Continue Learning"
  • • Extra Large (xl): min-h 60px, responsive text-h3, icon 28px, label: "Continue Learning"
  • • Full Width: Spans entire container width, maintains consistent height with medium size, ideal for prominent call-to-actions and form submissions

Secondary Button

Secondary Button

Purpose & Considerations

  • • Provides alternative or supplementary actions without pressure
  • • Reduced visual weight creates a non-pressuring experience
  • • Outline style maintains visibility while indicating optional nature
  • • Consistent interaction patterns with primary buttons for predictability

Size Variations

  • • Small (sm): 40x40px touchpoint, suitable for secondary actions in space-constrained layouts
  • • Medium (md): Default 44x44px touchpoint, balanced size for primary actions
  • • Large (lg): 48x48px touchpoint, increased prominence for key progression points
  • • Extra Large (xl): 52x52px touchpoint, maximum visibility for crucial journey milestones
  • • Full Width: Spans entire container width, maintains consistent height with medium size, ideal for form actions and secondary call-to-actions

Tertiary Button

Tertiary Button

Purpose & Considerations

  • • Enables content skipping and optional navigation choices
  • • Minimal styling reduces cognitive load and decision pressure
  • • Calm hover state provides gentle feedback without pressure
  • • Clear focus states maintain accessibility while being non-intrusive

Size Variations

  • • Small (sm): 40x40px touchpoint, minimal presence for auxiliary navigation
  • • Medium (md): Default 44x44px touchpoint, comfortable size for optional interactions
  • • Large (lg): 48x48px touchpoint, increased size for important but optional paths
  • • Extra Large (xl): 52x52px touchpoint, maximum size for prominent optional navigation
  • • Full Width: Spans entire container width, maintains consistent height with medium size, ideal for optional navigation and content skipping

States & Variations

  • • Default: Standard appearance for navigation or skip actions
  • • Hover: Calm blue background for gentle feedback
  • • Focus: Clear blue focus ring for accessibility
  • • Disabled: Muted color and reduced opacity for inactive state
